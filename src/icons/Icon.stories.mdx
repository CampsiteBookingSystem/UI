# Icons

These are the icons used inside the application.
Each icon is in `SVG` format so the color can easily be changed.
Any size can be chosen without affecting the quality of the icon.

## Usage

```jsx
import { Icon } from '@vulpee/ui';

<Icon className="MyCustomClass" color="var(--color-primary)" width="20" height="16" />;
```
