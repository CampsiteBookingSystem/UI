# Button

There are 2 categories of buttons you can choose from:

- **Default** which is a classic, rectangular button.
- **Rounded** which is a rounder button.

There are 3 variants of buttons for each category : **primary**, **secondary** and **danger**.

Each button is available in 3 sizes : **small**, **default** and **large**.

## Usage

```jsx
import { Button } from 'booking-system-ui';

<Button scope="default" variant="primary" onClick={handleClick}>
  Click here
</Button>;
```
